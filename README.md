# Vettrivel V | Cybersecurity Portfolio

This repository contains the source code for Vettrivel V's cybersecurity portfolio site.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting started

Make sure you have Node.js and npm installed.

```sh
npm install
npm run dev
```

The site will be available on the local development server shown in the terminal.
